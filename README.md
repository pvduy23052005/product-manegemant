# product-manegemant

## Giới thiệu

Ứng dụng quản lý sản phẩm xây dựng bằng Node.js theo mô hình **Server-Side Rendering**, hỗ trợ quản lý sản phẩm, đơn hàng và người dùng .

## Công nghệ sử dụng

- Express.js
- MongoDB + Mongoose
- Pug (Template Engine)
- Cloudinary (Upload ảnh)
- CSS, JavaScript
